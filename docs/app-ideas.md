# パーソナライズ音声合成のアプリケーションアイデア

## 着想

ずんだもんの成功は技術的な新しさではなく、アプリケーションとしての優れた設計にある。
- 動画解説で使いやすいキャラクター性
- 親しみやすい声とセリフ
- 人が「使いたくなる」体験設計

技術（TTS、Voice Cloning等）はもう揃っている。問題は「アプリケーションとして何が刺さるか」。

## コンセプト: ロックマンエグゼ的パーソナルナビ

PET = ターミナル、ナビ = Claude Code + 音声、オペレーター = ユーザー。

自分専用の相棒がターミナルの中にいて、声で状況報告してくれる。

### 方向性

- **自分だけの相棒の声** — 好きなキャラや自分の声でClaude Codeが喋る
- **状況に応じたセリフ変化** — テスト通ったら嬉しそうに、エラーなら焦り気味に
- **名前で呼んでくれる** — 「やたろう、できたのだ！」
- **ポモドーロタイマーの声かけ**
- **朝の天気・予定読み上げ**
- **git pushしたら「行ってらっしゃい」的な声**

## 掘り下げるべき問い（未整理）

- 「パーソナライズ」のどこに価値があるか（自分の声？好きなキャラ？状況適応？）
- 誰の・どんな課題を解決するのか
- アプリケーションとしての最小単位は何か
